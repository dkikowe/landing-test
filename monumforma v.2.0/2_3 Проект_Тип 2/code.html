Project #11 Hotel

Синтез архитектурного масштаба и камерной атмосферы гостеприимства.

Архив проектов chevron_right Project #11 Hotel
01 / Concept

Художественное видение

Интеграция монументальной скульптуры в современное пространство отеля требует деликатного баланса. Объект "Flowing Void" для Project #11 — это исследование взаимодействия жесткой металлической формы и мягкого интерьерного освещения.

Центральная идея заключалась в создании "визуального якоря" в лобби, который меняет свой характер в зависимости от времени суток, отражая динамику отельной жизни.

02 / Engineering

Инженерная лаборатория

Сложная геометрия потребовала 4-месячного цикла расчетов. Мы использовали параметрическое моделирование для оптимизации веса при сохранении структурной целостности 6-метровой консоли.

02.1 / Топологическая оптимизация

Снижение веса на 34% без потери жесткости.

02.2 / Расчет ветровых нагрузок

Анализ потоков для открытых зон атриума.

02.3 / Узлы сопряжения

Скрытые крепления с точностью до 0.1мм.

03 / Production

История производства

Процесс создания занял 120 рабочих дней. Мы использовали авиационный алюминий и аргоновую сварку для достижения бесшовной поверхности. Каждый элемент полировался вручную до состояния "жидкого металла".
